# Deerwalk's Delicious Delicacies

This is a simple static menu website created as a class assignment.  
The design showcases three main categories:

- **Breakfast**
- **Lunch**
- **A'La Carte**

Each section includes:
- Item name
- Price
- Short placeholder description
- Accompanying food image
